# Processus et services
## Processus
Un processus est une instance d'un programme en cours d'éxécution.

- Affiche les processus liés à la console ouverte
```bash
ps
```

- Affiche tous les processus de la machine
```bash
ps -aux
```

- Retrouver un processus particulier
```bash
ps -aux | grep [nom pressus rechercher]
```
